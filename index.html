Ubuntu 环境清理 清理无用的依赖 1#apt-get autoclean 清理孤儿安装包 1#apt-get autoremove 清理无用的配置 1#dpkg --list | grep "^rc" | cut -d " " -f 3 | xargs sudo dpkg --purge 删除手动安装的依赖包 ​ 虽然apt 提供的包的依赖关系的管理，apt-get autoremove 也可以清理掉包的依赖，但是当你安装包A之前已经手动安装了A的依赖包B，这样apt是没有办法 ​ 清理调包B的。所以我们必须借助另一个命令：deborphan 来完成清理。deborphan 不是系统自带命令需要自己安装: 1#apt-get install deborphan ​ 列出孤儿的安装包 1#deborphan ​ 清理掉无用安装包 1#deborphan | xargs sudo apt-get purge -y
